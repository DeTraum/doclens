# DocLens

本地隐私优先的「图片 / 文档 → 可编辑文档」工具。识别在本机完成，源文件不出设备。

## 三种模式

- **本地上传（截图 OCR）** — 拖图/粘贴截图，本机识别成 Markdown / Word，或上传飞书。可批量选文件夹。
- **飞书链接** — 粘贴含截图的飞书文档链接，提取图片本地 OCR，生成新的可编辑飞书文档（标题沿用源文档）。
- **公众号文章** — 粘贴公众号/网页链接，抓取原生内容高保真转飞书。支持批量（多个链接 / 上传 txt）。

## 能还原的内容

文字、**标题层级**、**有序/无序列表（含嵌套）**、**表格（含合并单元格）**、图片、代码块、**公式(LaTeX)**、分割线、引用块；公众号还支持 **加粗 / 斜体 / 下划线 / 删除线 / 颜色 / 高亮 / 超链接**。

> 截图来源受 OCR 限制：可还原文字/标题/表格/列表/公式/图片；内联样式（加粗/颜色等）只有公众号(HTML源)能保留。

## 快速开始

### macOS（推荐：双击）
1. 双击 **`安装.command`**（自动装 Python、依赖、下载模型，首次约 1-2GB）
2. 双击 **`启动.command`**（自动打开浏览器 http://127.0.0.1:8910）

> 首次双击被系统拦截时：右键该文件 →「打开」→ 再点「打开」。

### 命令行（Linux / 手动）
```bash
chmod +x setup.sh && ./setup.sh
source .venv/bin/activate
python main.py
```

## 飞书配置（要用飞书功能才需要）

详见 **`飞书配置图文教程.md`**（页面右上角「⚙️ 配置飞书」也有同款向导）。要点：
1. 飞书开放平台创建企业自建应用，记下 App ID / App Secret
2. 权限管理开通：`docx:document`、`drive:drive`、`bitable:app`
3. 安全设置加重定向 URL：`http://127.0.0.1:8910/api/auth/callback`
4. 版本管理与发布（必须发布权限才生效）
5. 页面里填凭证 → **点「登录飞书」**

> **必须登录**才能转飞书：保证生成的文档/管理表都归你账号所有、在你云盘、不公开。

## 管理与同步

每次转飞书的记录自动写入一张 **多维表格**（标题/链接/来源/时间）。登录同一飞书账号后，**换电脑也同步同一张表**，方便核对历史。页面右上角「📊 我的转换记录」直达。

## 隐私与安全

- 服务**仅监听 127.0.0.1**，外部访问不了（强制锁定，需显式 `DOCLENS_ALLOW_LAN=1` 才对外）
- 源图本机识别，识别完即删；只有你选「上传飞书」时识别结果才上云
- 已防护：路径穿越 / CSRF / SSRF；`data/` 内过期文件自动清理（默认7天，`DOCLENS_KEEP_DAYS` 可调）
- 飞书凭证仅存本机 `config.json`（已被 .gitignore 排除，打包分发自动剔除）

## 依赖

- Python 3.10 – 3.13
- [MinerU](https://github.com/opendatalab/MinerU)（OCR/版面识别，首次下载模型约 1-2GB）
- 其余见 `requirements.txt`

## 分发给别人

双击 **`打包分发.command`** 生成干净的 `DocLens_分发包.zip`（自动排除你的凭证/数据），发给对方解压后双击安装即可。
