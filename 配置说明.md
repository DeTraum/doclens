# DocLens 使用说明

图片/截图/公众号 → 可编辑飞书文档（或本地 Word/Markdown）。

---

## 一、安装（只做一次）

1. 双击 **`安装.command`**
   - 自动创建环境、安装依赖、下载 OCR 模型（约 1-2GB，首次较慢，耐心等）
   - 如果提示没装 Python：去 https://www.python.org/downloads/ 装好再重新双击

> 第一次双击 `.command` 若被 macOS 拦截：右键文件 →「打开」→「打开」。

## 二、启动

1. 双击 **`启动.command`**
2. 浏览器自动打开 `http://127.0.0.1:8910`
3. 要停止：关闭那个黑色窗口即可

---

## 三、配置飞书（只做一次，想用飞书功能才需要）

> 只用本地识别、下载 Word/Markdown 的，**跳过这步即可**（断网也能用）。

### 1. 创建你自己的飞书应用
1. 打开 [飞书开放平台](https://open.feishu.cn/app) → **创建企业自建应用**
2. 进入应用 → **「权限管理」**，开通这 3 个权限（搜索关键词即可）：
   - `docx:document`（创建编辑文档）
   - `drive:drive`（云空间）
   - `bitable:app`（多维表格，用于自动记录转换历史）
3. **「安全设置」→ 重定向 URL** 添加：
   ```
   http://127.0.0.1:8910/api/auth/callback
   ```
4. **「版本管理与发布」→ 创建版本 → 申请发布**（自己是管理员就直接通过）

### 2. 在 DocLens 里填凭证 + 登录
1. 页面「飞书链接」标签 → 展开「飞书应用配置」
2. 填入你应用的 **App ID** 和 **App Secret** → 保存并验证
3. 点右上角 **「登录飞书」** → 同意授权
4. 完成！以后生成的文档/表格**都归你所有、在你云盘里、可直接编辑**

---

## 四、三种用法

| 模式 | 用途 | 要联网吗 |
|---|---|---|
| **本地上传** | 截图/图片 → 文字+表格，下载 Word/MD | 否（可断网） |
| **本地上传 → 上传飞书** | 识别结果存进飞书，源图不出本机 | 是 |
| **公众号文章** | 公众号链接 → 飞书文档（图文表格高保真） | 是 |
| **飞书链接** | 飞书文档里的图 → OCR → 新飞书文档 | 是 |

小技巧：
- 长截图自动分段识别，不丢字
- 浅色水印：勾选「去图片水印(浅色)」
- 所有转换记录自动汇总到「📊 我的转换记录」表里，方便查找历史

---

## 五、隐私说明
- **本地上传**：源文件只在你电脑本机识别，识别完即删，可全程断网
- **上传飞书/公众号/飞书链接**：需要联网（内容要发给飞书生成文档）
- 服务只监听本机 `127.0.0.1`，外部无法访问
